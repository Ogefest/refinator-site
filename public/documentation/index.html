# Introduction

Refinator is no-code data engineering desktop application to simplify data cleansing and preparation. Using refinator you are able to:

  • perform simple text operations like substring, trim, replace, reformat on cell or column
  • work with columns even if file is not CSV (reorder,duplicate,remove etc)
  • add content to selected columns/rows
  • group edit for columns or cells
  • make conditional operations for particular rows
  • basic math operations, like add 42 to column value
  • mix functionalities and create recipes for repeated tasks

All of this you can do without bother about file size. You have 5GB file or 10M rows file? No worries, Refinerator handle that.
