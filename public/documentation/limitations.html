# Limitations

One of the strongest feature of Refinerator is processing on unlimited file size. Refinator architecture has not limit about file size and you can open 5GB file on PC with 4GB of RAM with small RAM usage footprint. All applied tasks are performed on your HDD, using SSD or NVMe is recommended.

# Known limitations because of Refinator architecture:

  • drive space - to open any file you need at least twice as much free space. To open 1GB file you need 2GB free space on your drive, to open 10GB file you need 20GB free space etc.

# Known limitations because of used libraries:

  • max file size - 281TB file
  • max column number - 1990 included columns with data and tags. For example 10 columns and 5 tags makes 15 columns used
  • max rows number - 2^64 or 18446744073709551616

If you will try to open 100GB file it should be fine but it take time, all depends on your hardware.
