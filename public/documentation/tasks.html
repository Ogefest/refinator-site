    # Tasks

    # Content

    Name Description
    Filter Filter rows, can keep, remove or add tag to particular rows
    Trim Remove white spaces from beginning and end of the column/row
    Replace Replace value with other value
    Change case Change case to lower, upper or camel
    Append Add prefix or suffix
    Insert Insert value, random string, random number, hash, UUID4
    Substring Cut value by index start and length
    Math Make math operations on value
    Reverse Reverse string
    Date reformat Change date formatting. Check Oracle docs for details (opens new window)

    # Column

    Name Description
    Split Split one column to more columns using separator
    Merge Merge two or more column into one using separator
    Reorder Change column order
    Remove Remove selected columns
    Insert Insert empty column after selected column
    Duplicate Insert new column with content from selected column

    # Rows

    Name Description
    Split Split row into two or more rows using separator
    Merge Merge rows into one row using separator
    Insert Insert empty row
    Remove Remove rows by tag
    Duplicate Duplicate rows
    Unique Unique rows in selected column
    Shuffle Randomize rows order

    # Task

    Name Description
    Row count Calculate number of rows with selected tag
    Go to line Scroll to file position
    Group edit Edit multiple values at once groupped by column or tag
    Search Search value in file
