# Privacy

Refinator respect your privacy. This application works on your computer in your network with your local files. There is no clouds or accounts.

# Refinator may connect to external servers in two cases:

  • send crash reports if occur. This is handled by Sentry (opens new window) and it send stacktrace to Sentry. In crash report there is no information about your files, it's only information where application stopped. This is useful to improve quality and fix bugs. You can turn off this functionality in application settings.
  • check if new version is available
